# AI-Powered Career Roadmap & Skill Gap Intelligence Platform

**B.Tech CSE Major Project** — Personalized career roadmaps powered by AI.

## Live Demo

🌐 Frontend: https://career-ai-frontend-1t7h.onrender.com

⚙️ Backend API: https://career-ai-backend-ikc1.onrender.com

📂 GitHub Repository: https://github.com/adityarajauriya29/Career-AI

## Features

* AI-powered career roadmap generation
* Resume parsing and skill extraction
* Skill gap analysis with readiness score
* AI career chatbot
* Role-based dashboards (Student, Mentor, Placement Officer, Admin)
* Progress tracking and analytics

## Stack

* **Frontend:** React 18, Tailwind CSS, React Router, Axios, Recharts
* **Backend:** FastAPI (Python 3.11), Motor (async MongoDB), Pydantic v2, JWT Authentication
* **Database:** MongoDB Atlas
* **AI:** Google Gemini, PyPDF, scikit-learn
* **Deployment:** Render + MongoDB Atlas
