# REST API Reference (excerpt)

Base URL: `/api`. All non-auth endpoints require `Authorization: Bearer <access_token>`.

## Auth
| Method | Path | Body | Returns |
|--------|------|------|---------|
| POST | /auth/register | {name,email,password,role} | tokens + user |
| POST | /auth/login | {email,password} | tokens + user |
| POST | /auth/refresh | {refresh_token} | access_token |
| POST | /auth/logout | — | message |
| POST | /auth/change-password | {old_password,new_password} | message |
| GET | /auth/me | — | user |

## Profile (student)
| GET | /profile | current profile |
| PUT | /profile | updated profile |

## Career Roles
| GET | /roles | list |
| GET | /roles/{name} | one |
| POST | /roles | admin only |

## Skill Gap
| POST | /gap/analyze | {target_role} → {matched,missing,readiness_percentage} |

## Roadmap
| POST | /roadmap/generate | {target_role,weekly_hours} |
| GET | /roadmap | current saved roadmap |

## Resume
| POST | /resume/upload | multipart PDF → {skills,certifications,projects} |

## Progress
| POST | /progress/update | {skill,completed} |
| GET | /progress | summary + items |

## Mentor (mentor/admin)
| GET | /mentor/students |
| GET | /mentor/student/{id} |
| POST | /mentor/suggest | {student_id,text} |

## Placement (placement_officer/admin)
| GET | /placement/overview |

## Chatbot
| POST | /chatbot/ask | {question,history} |
| GET | /chatbot/history |

## Admin
| GET | /admin/stats |
| GET | /admin/users |
