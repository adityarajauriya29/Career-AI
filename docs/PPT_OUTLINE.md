# Presentation Outline (15 slides)

1. **Title** — Project name, your name, guide, college
2. **Agenda**
3. **Problem Statement** — bullet points + 1 data stat
4. **Existing Systems** — short comparison table
5. **Proposed Solution** — 1 diagram
6. **Objectives** — 5 bullets
7. **System Architecture** — high-level diagram
8. **Tech Stack** — logos grid (React, FastAPI, MongoDB, Gemini, spaCy)
9. **Modules (11)** — icons grid
10. **AI Pipeline** — Resume → Skills → Gap → Roadmap (flow diagram)
11. **Screenshots** — Dashboard, Roadmap, Gap, Chatbot (4 thumbnails)
12. **Database Design** — collections diagram
13. **Testing & Results** — test count + key metrics
14. **Future Scope** — embeddings, mobile app, job feed
15. **Thank You / Q&A**

Design tips: dark theme with brand-blue accent, use icons (Lucide), 1 idea per slide.
